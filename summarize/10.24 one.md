**复习昨天**

```
1.模型定义
	AutoField 
	CharField
	IntegerField
	BooleanField
	DateTimeField
	DateField
2.约束
	primary_key
	unique
	default
	auto_now
	auto_now_add 用在记录第一次时间
3.定义表名
	class Mate:
		db_table="student"
	如果指定db_table参数，则在迁移后再数据库中的表名为student
	如果没指定db_table参数，在数据库中表名为应用app的名称_模型名小写
4.迁移
	1.python manage.py makemigrations
	生成迁移文件
	2.python manage.py migrate
    执行迁移到数据库中，特别是在修改约束条件后要执行一遍
5.模型的CRUD
	1.创建 
		-模型名称.objects.create(字段)
	2.查询
		-查所有 all()
		-查询满足条件 模型名.objects.filter().filter()
					 模型名.objects.filter(字段1，字段2)
		-获取确定的某一个
			-模型名.objects.get(条件)
			-注意：条件不满足 无法获取数据 则报错
			-满足条件数据多于1个对象，也报错
		-排除满足条件的数据
			-模型名.objects.exclude(字段)
		-返回结果序列化
			-模型名.objections.values(字段1，字段2)
			-如果括号里面没写，返回的是全部
		-排序
			-升序 模型名.objects.order_by(字段1)
			-降序 模型名.objects.order_by(-字段1)
		-查询过滤条件
			-字段__contains
			contains,startwith,endwith,gt,gte,lt,lte,
	3.删除
		-模型名.objects.filter(字段1).delete
		
	4.更新
		-模型名.objects.filter(字段1).update(字段1='新值')
	
	5.与或非Q()
		-& ，
		-|
		-~
	字段的比较
		from django.db.models import Q,F
		stus=Student.objects.filter(yuwen__gt=F('shuxue')+10)
	6.关联关系
		-一对一:OneToOneField
			class A:
				id=xxxx
				b=OneToOneField(B)
			class B:
				id=xxxx
				已知a对象，查找B：a.b
				已知b对象,查找A:b.a
				
				
		-一对多:Foreignkey
				class A:
				id=xxxx
				b=Foreignkey(B)
			class B:
				id=xxxx
				已知a对象，查找B：a.b
				已知b对象,查找A:b.a_set
		-多对多
```
